NodeFramework
=============

# DEVELOPERS

## INSTALL

You need NodeJS 0.11.9 as KoaJS do.

    $ (install NVM)
    
    $ nvm install 0.11.9
    $ nvm alias default 0.11.9
    $ nvm use default
    
    $ npm install # install dependencies

## TRY

You can try any test project available in src/app_test.

    $ cd src/app_test/{anyapp}
    $ node cli.js

NodeFramework
=============

## INSTALL

You need NodeJS 0.11.9 as KoaJS do.

    $ (install NVM)
    $ nvm install 0.11.9
    $ nvm alias default 0.11.9
    $ nvm use default
    $ npm install # install dependencies

Then try it out with a test project

    $ cd src/app_test/{anyapp}
    $ node cli.js
